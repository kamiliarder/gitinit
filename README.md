# gitinit
add your projects to your remote repository on github

## Prerequisite
- github account
- git

## WARNING! 
this script initializes all your directories inside the directory you ran the script in

### Usage
1. clone this repository on where you'll place your user scripts (or anywhere works)
3. create a new remote repository on github
4. now run the script inside your project directory
5. the script will ask for your github username and the name of the remote repository
